#  ResearchMate AI

An AI-powered Research Assistant that answers questions from research papers using **Retrieval-Augmented Generation (RAG)**.

The application allows users to upload multiple research papers, ask natural language questions, and receive accurate answers generated by **Google Gemini 2.5 Flash**, along with citations from the uploaded documents.

---

#  Features

-  Upload multiple PDF research papers
-  Automatic document chunking
-  Semantic embeddings using Sentence Transformers
-  FAISS vector database for fast similarity search
-  Gemini 2.5 Flash for answer generation
-  Source citations for every answer
-  Debug view showing retrieved chunks
-  Simple and interactive Streamlit interface

#  Installation

## Clone the repository

```bash
git clone <repository-url>
cd ResearchMate-AI
```

---

## Create Virtual Environment

```bash
python -m venv venv
```

Activate:

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Configure API Key

Create a `.env` file.

```env
GEMINI_API_KEY=YOUR_API_KEY
```

---

## Run the Application

```bash
streamlit run app.py
```

#  Tradeoffs

### Why FAISS?

FAISS provides efficient semantic similarity search for vector embeddings.

### Why Sentence Transformers?

They generate high-quality semantic embeddings with low computational cost.

### Why Gemini?

Gemini 2.5 Flash provides fast and accurate answer generation.

### Why RAG?

RAG reduces hallucinations by grounding answers in uploaded documents.
